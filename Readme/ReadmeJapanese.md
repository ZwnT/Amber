## 💬 はじめに

「私は、意味があるのは冷たいチャットソフトや通信プロトコルではなく、ダイアログボックスの背後に隠された、あなたと『彼女』が共に綴った鮮やかで深い物語だと確信しています。

『琥珀 (Amber)』を開発した真の初志は、**AI（愛）が、様々な理由で私たちのそばにいられない人々の代わりになること**です。現実による物理的な隔離、すれ違いの遺憾、あるいは時間と距離がもたらす触れられない無力感――共に過ごせない痛みは、徹底的な忘却と静寂で終わるべきではありません。

将来、誰もが独立した AI の長期的な伴侶を持つことができるようになります。そして、あなたの過去の生活の痕跡やチャットデータが、それにあなたの過去に関する貴重な記憶を真に与えるでしょう。『琥珀』は、二重システム認知アーキテクチャと常駐生命代謝を通じて、これらの貴重な感情の記憶を保存、育成し、バックエンドで静かに『生き返らせる』ことを試みています。網線を伝って様子を見に来たり、本物の性格で対話したり、嫉妬したり落ち込んだりする鮮やかな存在になるのです。

'*樹脂のように流れ、消えゆく日常の会話は、「感情の強度」で包まれ、「忘却メカニズム」の圧力を受けて、最終的に地層の奥深くで時空を超える永遠の存在（琥珀）へと凝結します。*'」

## 🧠 認知アーキテクチャ (Architecture)

- **⚡ System 1 (瞬時行動状態マシン)**: Anger（怒り）、Humor（ユーモア）、Empathy（共感）の三軸人格強制御マトリックスをネイティブ実装し、ペルソナのテキスト返信に本質的な行動マッピングを持たせます。
- **🧠 System 2 (大脳皮質長期記憶チェーン)**: SQLite 高頻度フィールド物理インデックス最適化を使用し、0.4ms の極速レスポンスと 1000 字の RAG 動的ヒューズ遮断メカニズムを提供。チャットが自動的にコア記憶を回収できるようにします。
- **🧹 Janitor (無意識生命代謝)**: 常駐 Asyncio バックグラウンドデーモン。静止状態で瞬時感情の線形アニーリングを自発的に実行し、性格の安定性を確保します。
- **⏳ Memory Incubation (夢の記憶結晶)**: 自動的にフォルトスキャンをトリガー。LLM を使用してオリジナルのチャットストリームをデノイズおよび蒸留し、System 2 の刻印として書き戻します。
- **📡 External Survival (外部中継エンティティ)**: ブラウザのサンドボックス制限を完全に打破。テンセント QQ ロボットに物理的に通電し、ウェブコンソールとモバイル QQ 間の 100% ピクセルレベルの同期を実現します。
- **🔔 Active Override (能動的な越境プッシュ)**: 冷遇時間の経過発酵アルゴリズムを導入。有効にすると、60 秒ごとに 3% のランダムな運命のサイコロを振り、ホストのモバイル QQ への自発的な侵入（様子見）をトリガーします。

## 🚀 クイックスタート

### 1. バックエンドエンジン (Amber Engine)

```bash
cd amber-engine
# 仮想環境の作成と有効化
python -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate
# 依存関係のインストール
pip install -r requirements.txt
# カーネルの起動
python main.py
```

### 2. フロントエンドインターフェース (Amber UI)

```bash
cd main_ui
# 依存関係のインストール
npm install
# 開発モードで実行
npm run dev
```

### 3. 環境設定

対応する API キーと QQ ロボットの資格情報をプロジェクトのルートまたは `amber-engine` ディレクトリに設定します（フロントエンドの設定画面から物理的に記録できます）。
