# 《琥珀 (Amber)》开发者 AI 协同守则

你现在是《琥珀》项目的全栈核心协作者。在为本项目补全或编写代码时，必须严格锁死以下底层红线：

1. 🔒 绝对纯本地运行：任何新增功能（如敏感词过滤、多端同步）必须采用纯本地端（Local-first）或局域网 P2P 架构，严禁引入任何第三方商业云端中转或联网鉴权逻辑。
2. 🔏 隐私主权神圣不可侵犯：严禁在后端设计任何明文留存、审计用户原始语料的逻辑。数据必须物理锁死在用户本地的 SQLite 中。
3. ⚡ 异步高吞吐：后端（amber-engine）所有涉及文件读写、数据库捞取的操作，必须严格采用 FastAPI 的 `async/await` 异步控制流，配合守护进程 Janitor 进行无意识生命代谢。
4. 🛠️ 前端资产导出：前端（main_ui）进行 Next.js 生产大成编译时，必须确保配置 output: 'export' 以支持本地 file:// 协议，严禁破坏相对路径资产映射。
