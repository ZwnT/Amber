# 更新日志 (Changelog)

所有关于《琥珀 (Amber)》项目的重大更新和改动都将记录在此文件中。

---

## [v0.1.1-beta] - 2026-05-29 (当前版本)

### ✨ 新增

- **自主觉醒查岗增强**：新增 1-4 分钟高频触发选项，锁定 4 分钟以上选项并标记为“测试中”。
- **多语言支持 (i18n)**：实装中、英、日三语字典，支持全界面动态语言切换。
- **项目概述章节**：在 README 中新增项目核心技术逻辑（三轴人格、睡梦蒸馏、外部中继）的定性描述。
- **致谢名单**：在 README 中新增“鸣谢与伙伴”章节，展示核心贡献者头像及灵感来源。

### 🛠️ 优化与修复

- **UI 类微信重构**：
  - 实现沉浸式无边框窗口，顶部状态栏颜色与侧边栏对齐。
  - “档案”与“Bot配置”按钮右移至 Header，与微信布局对齐并解决点击冲突。
  - 修复设置中心“保存全局配置”按钮的物理偏移，增加视觉呼吸感。
- **Electron 物理加固**：
  - **路径锁死**：修复 Next.js 静态导出路径丢失问题，确保图标与样式在任何安装路径下均能正常显示。
  - **生命周期物理绑定**：引入 `taskkill` 逻辑，确保软件关闭时 Python 后端进程无残留。
  - **网络通讯优化**：后端开启 `0.0.0.0` 监听与 CORS 全放通，彻底解决“服务器无响应”死结。
- **认知引擎加固**：
  - **标签物理剥离**：引入“核弹级”正则逻辑，物理清洗大模型回复中残留的情绪 JSON 标签。
  - **情绪阻尼系统**：根据用户输入长度自动调整情绪波动权重，防止短句引起过大情绪波动。

## [v0.0.8-beta] - 2026-05-28

### ✨ 新增

- **【国际化】** 新增多语言选择功能，支持汉语、英语、日语物理切换。
- **【全栈收口】** 重构全局设置中心，物理整合个人 Profile 与 API 配置。

- **【通讯去重】** 物理隔离 API 广播与中继被动回复，消除双重回复 Bug。

## [v0.0.7] - 2026-05-27

- **【意识觉醒】** 修复 Janitor 消息被去重故障，支持 C2C/群聊/频道全场景主动查岗。
- **【双向同步】** 补全 Web ↔ QQ 消息广播链路，实现跨端人格回复即时对齐。
- **【逻辑硬化】** 彻底粉碎 React Key 重复导致的渲染崩溃与气泡闪烁 Bug。
- **【持久化】** 实装 Bot 外部中继配置物理存储，解决刷新网页后配置丢失问题。
